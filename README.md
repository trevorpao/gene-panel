GeneJs Panel
===========

## Documentation

### Install
```
    $ npm install
    $ npm install --only=dev
    $ "0.0.0.0  f3cms.lo" >> /etc/hosts
```

### DEV
```
    $ npm run dev #to preview and watch for changes
    $ open http://f3cms.lo:8008/
```

### Deploy
```
    $ npm run build #to build your webapp for production
```
