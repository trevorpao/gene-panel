## Install
> 
> nvm install v14.17.4
> 
> npm i
> 
> npm i --global webpack-cli
> 
> "0.0.0.0  loc.f3cms.com" >> /etc/hosts
> 

## DEV
> 
> nvm use v14.17.4
> 
> npm run dev # to preview and watch for changes
> 

## Deploy
> nvm use v14.17.4
> 
> npm run build #to build your webapp for production
